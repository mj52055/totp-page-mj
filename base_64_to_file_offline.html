说明：
1. 支持完整 Data URL，例如 data:image/png;base64,...。
2. 也支持纯 Base64。纯 Base64 会尝试通过文件头自动判断 PNG、JPG、GIF、WEBP、PDF、ZIP、文本等格式。
3. 如果自动识别不准确，可以在右上角手动选择文件类型。
4. 页面不依赖任何外部 JS、CSS 或网络资源，可直接放在内网或本地打开。
